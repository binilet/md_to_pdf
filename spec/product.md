# Markdown file to multiple file formats converter(PDF, HTML, DOCX, EPUB)

## Goal
Provide a web interface to convert Markdown text into a downloadable PDF (PDF is Phase one).

## MVP Features
1. Markdown editor
2. Live preview
3. Export to PDF button
4. SIGN IN OR REGISTER BUTTONS

## Layout

+---------------------------+
| Toolbars                   |
+-------------+-------------+
| Editor      | Preview     |
|             |             |
+-------------+-------------+

## User Flow

1. User writes or uploads markdown file
2. Preview updates live
3. User clicks "Export PDF"
4. Markdown sent to backend
5. PDF downloaded

## Non-goals
the following features are not in scope for now:
- authentication
- document storage
- multi-user support
